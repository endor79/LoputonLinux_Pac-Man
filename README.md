# Hakkında
LoputonLinux'un Pac-Man sürümü, USB üzerinden offline ve online oyunu oynamak için tasarlandı.

# Özellikler
* 100'e yakın oyun bulunuyor.
* RTW89 WIFI ve BT sürücüleri eklendi.
* Devuan 5.0 tabanlı.
* 6.5.12 Xanmod Kernel.
* Xfce 4.18.
* UEFI desteği.
* Live Boot.
